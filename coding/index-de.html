Mission 3: Wie Computer funktionieren

Computer sind außergewöhnliche Maschinen, die Informationen verarbeiten, indem sie präzisen Anweisungen folgen.

Meister Luciano sagt: "Stellt euch den Computer als unser Missionskontrollzentrum vor: Er empfängt Informationen, verarbeitet sie und erzeugt Ergebnisse!"

Hauptkomponenten eines Computers:

CPU – Das Gehirn des Computers

Die CPU (Central Processing Unit) ist das Gehirn des Computers und führt die Anweisungen von Programmen aus.

Arbeitsspeicher – Der Arbeitstisch

Der RAM-Arbeitsspeicher ist wie ein Arbeitstisch, auf dem der Computer die Informationen ablegt, die er gerade verwendet.

1. Was macht die CPU in einem Computer?
A. Zeigt Bilder auf dem Bildschirm an.
B. Führt die Anweisungen von Programmen aus.
C. Speichert Dateien langfristig.
2. Welche Computerkomponente ist wie ein Arbeitstisch?
A. Die Festplatte.
B. Der RAM-Arbeitsspeicher.
C. Die Grafikkarte.

Mission 5: Bedingte Anweisungen

Bedingte Anweisungen ermöglichen es dem Computer, Entscheidungen basierend auf bestimmten Bedingungen zu treffen.

Meister Luciano sagt: "Bedingungen sind wie Weggabelungen während einer Erkundung: Je nachdem, was dir begegnet, entscheidest du, welchen Weg du nimmst!"

Die Geschichte der Algorithmen

Das Wort „Algorithmus“ leitet sich vom Namen des persischen Mathematikers Al-Chwarizmi ab, der im 9. Jahrhundert lebte. Seine Bücher führten die indischen Zahlen (die wir heute verwenden) und Methoden zur Lösung von Gleichungen ein.

Im Laufe der Jahrhunderte haben viele Mathematiker zur Entwicklung von Algorithmen beigetragen, aber mit dem Aufkommen von Computern wurden Algorithmen grundlegend für unser tägliches Leben.

Wusstest du schon? Der erste Computer-Algorithmus wurde von Ada Lovelace im Jahr 1843 für die Analytische Maschine von Charles Babbage geschrieben, lange bevor es moderne Computer gab!

Wie funktioniert ein Compiler?

Ein Compiler ist ein spezielles Programm, das Code, der in einer Programmiersprache (wie Python oder Java) geschrieben ist, in Maschinensprache übersetzt, die der Computer direkt verstehen und ausführen kann.

Der Kompilierungsprozess erfolgt in mehreren Phasen:

  1. Lexikalische Analyse: Der Code wird in „Tokens“ (Schlüsselwörter, Bezeichner usw.) zerlegt.
  2. Syntaktische Analyse: Überprüft, ob die Tokens korrekt nach den Regeln der Sprache organisiert sind.
  3. Semantische Analyse: Stellt sicher, dass der Code logisch sinnvoll ist.
  4. Codegenerierung: Erzeugt den Code in Maschinensprache.

Meister Luciano sagt: "Stell dir den Compiler als einen universellen Übersetzer vor, der deine Ideen in eine Sprache übersetzt, die der Computer verstehen kann!"

Wichtige Frauen in der Technologie

Neben Ada Lovelace und Grace Hopper haben viele andere Frauen grundlegende Beiträge zur Informatik und Technologie geleistet:

  • Hedy Lamarr: Schauspielerin und Erfinderin, entwickelte eine Technologie, die Grundlage für Wi-Fi und Bluetooth ist.
  • Radia Perlman: Bekannt als „Mutter des Internets“, erfand das Spanning-Tree-Protocol.
  • Anita Borg: Gründete das Institute for Women and Technology und die Grace Hopper Celebration.
  • Margaret Hamilton: Leitete die Entwicklung der Software für das Apollo-Programm der NASA.
